# Good Examples

Use these example shapes to anchor storyboard decisions.

## Decision Framework Explainer

- Open with the single question the framework answers.
- Animate the framework's core unit or relationship first.
- Use later scenes to show why that framing changes decisions.

## Diagram Walkthrough

- Start with the full system in a quiet state.
- Highlight one path, node, or phase at a time.
- Use labels sparingly and keep the camera fixed unless movement is necessary.

## Comparison Video

- Hold a stable frame and swap only the parts that change.
- Make the comparison rule obvious before showing the verdict.

## Process Or Lifecycle Video

- Use a single progress metaphor through the whole piece.
- Give each stage one sentence of explanation and one visual state change.
