# Scenario Selection Guide

Choose one primary scenario before drafting the storyboard.

## Framework Explainer

Use when the article introduces one core model, concept, or decision lens.

- Lead with the framing question.
- Reveal the framework structure in one to two scenes.
- Use later scenes to show why the framework changes the reader's judgment.

## Diagram Walkthrough

Use when the article depends on a flow, architecture, or staged movement through a system.

- Keep the camera steady.
- Highlight one path or node group at a time.
- Use path traces, node emphasis, and caption reveals.

## Comparison Explainer

Use when the article contrasts tools, states, or approaches.

- Keep the frame structure stable.
- Swap only the changing elements.
- Make the evaluation rule explicit.

## Process Or Lifecycle Walkthrough

Use when the article explains repeated steps, phases, or loops.

- Use a single progress cue throughout the video.
- Keep each stage to one visual transition and one short explanation.
- End with the consequence or decision the process supports.
