# Animation Recipes

Use these recipes when a storyboard scene needs a known-good motion pattern.

## Label Reveal

- Fade and translate labels in with short staggered timing.
- Hold long enough for full reading before the next state change.

## Node Emphasis

- Raise scale slightly and brighten contrast on the active node.
- Dim inactive nodes instead of moving the whole diagram.

## Path Trace

- Reveal a flow line progressively while related labels appear one beat behind.
- Use this for system flow, pipeline stages, or decision handoffs.

## Comparison Swap

- Slide or crossfade between two competing states while keeping the frame structure fixed.
- Use this for before versus after, framework comparisons, or trade-off scenes.

## Stage Stepper

- Move one clear indicator through a sequence while each stage gets a short explanation.
- Use this for lifecycle, process, or evaluation loops.
