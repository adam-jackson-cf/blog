# Research Note Capture Extension

## Load the extension

1. Run `bun run capture:service` from the blog repo.
2. Open `chrome://extensions`.
3. Enable Developer mode.
4. Click `Load unpacked`.
5. Select `/Users/adamjackson/Projects/blog/browser-extension`.

## Use it

- Open any normal `http` or `https` page.
- Click the extension action, use the keyboard shortcut, or use the context menu item.
- Review the prefilled note.
- Save the note to write content under `content/research-notes/`, assets under `assets/research-notes/`, and raw capture files under `.cache/research-notes/`.
